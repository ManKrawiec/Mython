# Mython Support

Lokalne rozszerzenie VS Code dla języka `Mython`.

## Co daje

- rozpoznawanie plików `.ps`
- podstawowe kolorowanie składni
- konfigurację komentarzy i nawiasów

## Jak użyć lokalnie

1. Otwórz projekt `Mython` w VS Code.
2. W razie potrzeby uruchom `Developer: Install Extension from Location...`.
3. Wskaż folder `vscode-mython`.

Po instalacji VS Code rozpozna pliki `.ps` jako `Mython`.
